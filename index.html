运费计算规则

运费计算公式：

运费 = 首重价格 + (计费重量 - 1) × 对应区间单价

重量区间说明：

  • (1,50]：1kg < 重量 ≤ 50kg
  • (50,100]：50kg < 重量 ≤ 100kg
  • (100,300]：100kg < 重量 ≤ 300kg
  • (300,500]：300kg < 重量 ≤ 500kg
  • (500,+∞)：重量 > 500kg

注：起重为30kg，运费最低收费按起重30kg计算
